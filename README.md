# FastFileSwitcher(FFS)
Is a Visual Studio Extension to switch fast between different files

## Features
 - [x] Add Files
 - [x] Display added files
 - [x] Delete Files from the switcher
 - [x] Switch between files
 - [x] Full Reset the Switcher
